Tryon Framework
Es un interfase de documento múltiple (MDI) diseñado para ser usado en páginas web (si, en HTML: php, .net, Java). Lo he hecho de uso sencillo.
 
Las ventanas creadas son movibles, se pueden cambiar de tamaño y se pueden apilar una sobre otra. Al seleccionar una ventana ésta se posiciona adelante. Puedes abrir tantas ventanas como necesites. Cada una es una página independiente. Soporta pantallas táctiles. Probado en Chrome, Firefox, Opera y IE.
PRE-REQUISITOS: Necesitas incluir jquery y jqueryUI en la página para que funcione.
 
Como usar el Framework Tryon:
 
a) Poner en la sección header las líneas necesarias para incluir jquery y jqueryUI (como usualmente harías):
<script src="http://code.jquery.com/jquery-latest.min.js"></script>
<script src="http://ajax.googleapis.com/ajax/libs/jqueryui/1/jquery-ui.min.js"></script>
<link rel="stylesheet" href="https://ajax.googleapis.com/ajax/libs/jqueryui/1/themes/smoothness/jquery-ui.css" />
 
b) Incluye el framework: descárgalo de aquí
<script src="scripts/tryonframework.js"></script>
 
c) Y ya está! Para abrir una ventana MDI, llamar a la función de la siguiente manera:
// top.popup(texto_tu_URL,texto_titulo,numero_izquierda,numero_arriba,numero_alto,numero_ancho);
top.popup('demo.html','titulo de la ventana',100,100,300,400);
