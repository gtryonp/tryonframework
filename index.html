Tryon Framework
Is a Multiple Document Interfase (MDI) framework designed to be used on webpages (yes, HTML: php, .net, Java). I made it simple as possible.
 
The created windows are fully movable, stackable, resizable. It moves to front when clicked. You can open all the windows as needed. Each document is opened as a new page, so each one is independient from another. The windows are fully touch enabled. Tested in Chrome, Firefox, Opera and IE.
PRE-REQUISITES: You will need jquery and jqueryUI to put it to work.
 
How to use the Tryon Framework:
 
a) Put in the header section the lines to include jquery & jqueryui (as usually you do):
<script src="http://code.jquery.com/jquery-latest.min.js"></script>
<script src="http://ajax.googleapis.com/ajax/libs/jqueryui/1/jquery-ui.min.js"></script>
<link rel="stylesheet" href="https://ajax.googleapis.com/ajax/libs/jqueryui/1/themes/smoothness/jquery-ui.css" />
 
b) Include the framework: download it here
<script src="scripts/tryonframework.js"></script>
 
c) That's all. To open a new MDI window, just call the function:
// top.popup(string_your_URL,string_window_title,integer_left,integer_top,integer_height,integer_width);
top.popup('demo.html','window title',100,100,300,400);
